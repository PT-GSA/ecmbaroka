Debug Admin Login

1. Cek Server Status

Pastikan server Bun berjalan:

bun run dev

Server harus berjalan di http://localhost:3000

2. Cek Database User

Jalankan script SQL di Supabase Dashboard:

-- Cek semua user SELECT id, email, created_at FROM auth.users ORDER BY created_at DESC; -- Cek user profiles SELECT id, full_name, role, created_at FROM user_profiles ORDER BY created_at DESC;

3. Buat/Update Admin User

Jika user belum ada sebagai admin, jalankan:

-- Ganti dengan user ID yang benar UPDATE user_profiles SET role = 'admin', full_name = 'Admin Susu Baroka' WHERE id = 'your-user-id';

4. Test Login

Buka http://localhost:3000/login

Login dengan email dan password yang sudah dibuat

5. Test Admin Access

Setelah login, akses:

  • http://localhost:3000/admin
  • http://localhost:3000/admin/dashboard

⚠️ Troubleshooting

  • Jika masih redirect ke login: Pastikan user sudah login dan role adalah 'admin'
  • Jika error 404: Pastikan file app/(admin)/page.tsx ada
  • Jika error database: Jalankan script SQL di atas
  • Cek console browser untuk error JavaScript

6. Debug dengan Browser

Buka Developer Tools (F12) dan cek:

  • Console tab untuk error JavaScript
  • Network tab untuk request yang gagal
  • Application tab untuk cookies dan session
