## üë• Manajemen Affiliate

### 1. Approve Aplikasi Affiliate Baru

#### Langkah-langkah:
1. **Login ke Admin Dashboard**
   - Akses `https://susubaroka.com/admin`
   - Login dengan akun admin

2. **Buka Halaman Affiliates**
   - Klik menu "Affiliates" di sidebar
   - Lihat daftar aplikasi pending

3. **Review Aplikasi**
   - Cek data aplikasi: nama, email, telepon
   - Review alasan menjadi affiliate
   - Cek pengalaman marketing (jika ada)

4. **Approve/Reject**
   - **Approve**: Klik "Approve" ‚Üí Set status "active"
   - **Reject**: Klik "Reject" ‚Üí Berikan alasan penolakan

#### Kriteria Approval:
```
‚úÖ DITERIMA:
- Data lengkap dan valid
- Email aktif dan dapat dihubungi
- Alasan yang masuk akal
- Tidak ada riwayat spam/fraud

‚ùå DITOLAK:
- Data tidak lengkap
- Email tidak valid
- Alasan tidak jelas
- Riwayat spam/fraud
```

### 2. Monitor Performa Affiliate

#### Dashboard Analytics:
1. **Total Affiliates**: Jumlah affiliate aktif
2. **Top Performers**: Affiliate dengan komisi tertinggi
3. **Conversion Rate**: Rata-rata konversi semua affiliate
4. **Total Commission**: Total komisi yang dibayarkan

#### Metrik Per Affiliate:
- **Total Klik**: Jumlah klik link affiliate
- **Total Konversi**: Jumlah pembelian yang dihasilkan
- **Total Komisi**: Komisi yang didapat affiliate
- **Conversion Rate**: Persentase konversi
- **Last Activity**: Aktivitas terakhir affiliate

### 3. Kelola Data Affiliate

#### Update Informasi:
- **Nama**: Update nama affiliate
- **Email**: Update email kontak
- **Telepon**: Update nomor telepon
- **Status**: Active/Inactive/Suspended
- **Commission Rate**: Custom rate komisi (opsional)

#### Suspend Affiliate:
- **Alasan**: Spam, fraud, atau pelanggaran
- **Durasi**: Temporary atau permanent
- **Notifikasi**: Kirim email ke affiliate

## üí∞ Manajemen Komisi dan Withdrawal

### 1. Monitor Komisi Affiliate

#### Real-time Tracking:
- **Pending Commission**: Komisi yang belum dibayar
- **Paid Commission**: Komisi yang sudah dibayar
- **Total Commission**: Total komisi affiliate
- **Commission History**: Riwayat komisi per periode

#### Commission Calculation:
```
Susu Baroka 200ml: 5% komisi
Susu Baroka 500ml: 7% komisi  
Susu Baroka 1L: 10% komisi

Contoh:
Order: 2 botol 500ml @Rp 25.000 = Rp 50.000
Komisi: 7% √ó Rp 50.000 = Rp 3.500
```

### 2. Kelola Withdrawal Requests

#### Langkah-langkah:
1. **Buka Halaman Withdrawals**
   - Klik menu "Withdrawals" di sidebar
   - Lihat daftar request pending

2. **Review Request**
   - Cek saldo komisi affiliate
   - Verifikasi data bank
   - Pastikan minimum withdrawal (Rp 50.000)

3. **Approve/Reject**
   - **Approve**: Set status "approved" ‚Üí Transfer
   - **Reject**: Set status "rejected" ‚Üí Berikan alasan

#### Status Withdrawal:
- **Pending**: Menunggu approval
- **Approved**: Disetujui, siap transfer
- **Processing**: Sedang diproses transfer
- **Completed**: Transfer selesai
- **Rejected**: Ditolak dengan alasan

### 3. Proses Pembayaran

#### Jadwal Pembayaran:
- **Harian**: Review request withdrawal
- **Mingguan**: Transfer setiap Senin
- **Bulanan**: Laporan komisi dan withdrawal

#### Metode Pembayaran:
- **Bank Transfer**: Transfer ke rekening affiliate
- **E-wallet**: OVO, GoPay, DANA (opsional)
- **Cek**: Untuk withdrawal besar (opsional)
